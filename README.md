Composerインストール用修正プラグイン
================================

本プラグインについて
-----------------

[Boilerplate](https://github.com/ec-cube2/boilerplate) でインストールしたEC-CUBE2用の挙動修正プラグインです。


修正項目
-------

以下の挙動を修正します。

管理画面

- 郵便番号インストール
- バックアップ
- phpinfo 無効化

EC-CUBE2 CLI
------------

以下のコマンドを追加します。

`fix-composer`
